# Мое портфолио

В этом репозитории собраны примеры моих работ как коммерческого и технического писателя. Для каждого примера в портфолио указан тип работы.

## Типы работ

- Руководство пользователя;
- Ответы на часто задаваемые вопросы;
- Статья.
